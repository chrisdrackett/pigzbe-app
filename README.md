# Pigzbe App

## Installation

```sh
brew install watchman
yarn global add react-native-cli
```

```sh
yarn install
```

```sh
react-native link
react-native link react-native-randombytes
```

```sh
yarn run game
```

## Usage

### Run Desktop App

```sh
yarn run desktop
```

### Run iOS App

```sh
yarn run ios
```

### Run Android App

```sh
yarn run android
```

### Run Linter

```sh
yarn run lint
```

### Run Tests

```sh
yarn test
```

## Content Updates

### Get latest content strings from CMS

```sh
yarn run content
```

### Rebuild game bundle for Webview

```sh
yarn run game
```
